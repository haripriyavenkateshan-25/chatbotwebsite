# Rule-Based Chatbot Website Using Python and Flask

## Project Overview

This project is a Rule-Based Chatbot developed using Python and Flask. The chatbot interacts with users through a web interface and responds to user queries based on predefined rules and pattern matching techniques. It is designed to simulate a simple conversation and provide information on various topics such as programming, technology, education, motivation, sports, and general knowledge.

Unlike AI-powered chatbots, this chatbot does not learn from data. Instead, it uses conditional statements and keyword matching to generate appropriate responses.

## Working Principle

1. User enters a message in the chatbot interface.
2. Flask receives the user input.
3. The chatbot converts the input into lowercase format.
4. The chatbot checks predefined rules using if-else statements.
5. A matching response is generated.
6. The response is displayed on the webpage.
7. The process repeats until the user exits the chat.
